# VIDEO BOOM BOT

Telegram бот, который добавит любой мемный звук или взрыв в ваш кружок

## Предыстория

Бот разрабатывался ещё осенью 2024 как просто добавление взрыва в конец кружка, позже в 2025 функционал увеличивался.

## Функционал

- Добавить взрыв (секундное видео) в конец кружка

- Добавить звук, доступный в боте в конец кружка

- Поиск мемных звуков (парсер сайта с красными кнопками)

- Добавление своего звука (могут пользоваться и остальные пользователи)

- Управление своими звуками

Отдельно отмечу систему sharing'а звуками. Когда пользователь находит нужный или загружает, ему выдается ссылка в формате ```https://t.me/video_boombot?start={ИДЕНТИФИКАТОР_ЗВУКА} ```
Перейдя по этой ссылке, у другого пользователя автоматически выберется этот звук.


- Популярные звуки
- Топ звуков

## Техническая сторона

**Stack**: Python Aiogram, PostgreSQL, Docker, moviepy, aiohttp, beautifulsoup4

Бот имеет собственную базу данных со звуками, которые использует, парсит сайт myinstants.com, чтобы реализовать популярные/поиск. Топ составляется локально.

## Как поднять у себя?

1. Скопируйте репозиторий:
    ```git clone https://github.com/zaamilion/boombot.git```
2. Создайте .env файл:
    ```
    TELEGRAM_API_KEY = ...
    POSTGRESQL_USER = "admin"
    POSTGRESQL_PASSWORD = "password"
    POSTGRESQL_HOST = "db"
    POSTGRESQL_PORT = "5432"
    POSTGRESQL_DBNAME = "postgres"
    BOT_URL= ...
    ```
3. Запустите:
    ```docker compose up -d --build```
